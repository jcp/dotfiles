# Installation

Download dotfiles repository.

```bash
cd ~/.config
git clone https://github.com/jcp/dotfiles.git
```

Execute `install.sh`.

```bash
cd dotfiles
chmod +x install.sh
./install.sh
```

> **Note**
>
> [Zsh and Oh My Zsh](https://github.com/jcp/installation-guides#terminal) must be installed.
