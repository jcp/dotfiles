# Installation

Download dotfiles directory.

```bash
$ cd ~
$ git clone https://github.com/jcp/dotfiles.git
```

Execute **setup.sh**.

```bash
$ cd dotfiles
$ chmod +x setup.sh
$ ./setup.sh
```

Cleanup and remove dotfiles directory.

```bash
$ cd ..
$ rm -rf dotfiles
```

Reload ZSH.

```bash
$ source ~/.zshrc
```
