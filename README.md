# Installation

Download dotfiles directory.

```bash
$ cd ~
$ git clone https://github.com/jcp/dotfiles.git
```

Execute `install.sh`.

```bash
$ cd dotfiles
$ chmod +x install.sh
$ ./install.sh
```

Cleanup and remove dotfiles directory.

```bash
$ cd ..
$ rm -rf dotfiles
```

Reload Zsh.

```bash
$ source ~/.zshrc
```

> **Note**
> 
> [Zsh and Oh My Zsh](https://github.com/jcp/installation-guides#terminal) must be installed.
